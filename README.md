# Instruktioner

- Starta WSL
- cd eller cd ~ för att komma till hemkatalogen
- cd code
- cd webb-komponenter
- code . för att starta vscode

För att skapa mappen webb-komponenter:
```bash
cd 
mkdir code
cd code
mkdir webb-komponenter
cd webb-komponenter
```

Annars skapa ett repo på github.
Kör Git clone på det.

```bash
git clone <url>
```
